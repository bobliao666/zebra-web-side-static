.css(name,value)

使用该方法可以设置一个或多个html元素的样式属性，或者获取某一个元素的某个样式属性，接收两个参数，一个名称，一个值（可选），返回string/object。

说明:

例如有以下代码：

<div id="getMyStylePrototype" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 获得我的样式 </div>

如上所示，获得id为'getMyStylePrototype'的'background-color'：

getMyStylePrototype function(){
    alert(zZ('#getMyStylePrototype').css('background-color'));
}

如上文代码所示，当您想要获得某个元素的样式属性的时候，可以在css()方法里填入一个样式属性名称，它将返回一个结果给你。

但是当我需要设置它的样式该怎么办呢，我们先看看下列代码：

<div class"setAll" id="setMyStylePrototype" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 设置我的样式 </div>
<div class"setAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 设置我的样式 </div>
<div class"setAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 设置我的样式 </div>
<div class"setAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 设置我的样式 </div>
<div class"setAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 设置我的样式 </div>

现在我们先把id为'setMyStylePrototype'的元素的背景颜色换掉看看:

setMyStylePrototype function(){
    zZ('#setMyStylePrototype').css('background-color','#000');
}

运行以上代码你将看到id为"setMyStylePrototype"的元素背景色改为黑色的效果。

又或者我一次性改变多个属性:

setMyStylePrototype function(){
    zZ('#setMyStylePrototype').css('background-color','#ccc')
    .css('border','3px solid #f70202')
    .css('width','200px');
}

需要注意的是，在填写.css(name,value）‘name’这个参数的时候，请保持属性名称和在编写真实网页的写法为一致！

我们也可以设置一组的样式属性:

setMyStylePrototype function(){
    zZ('.setAll').css('background-color','#ccc');
}

在设置属性的时候，选择器中所有存在的元素将会被设置上您所指定的属性值。

移除css属性:

setMyStylePrototype function(){
    zZ('.setAll').css('background-color','remove');
}

上文所示代码将移除所有class名为setAll的'background-color'
