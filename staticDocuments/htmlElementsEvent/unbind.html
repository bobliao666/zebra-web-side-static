.unbind(eventName,function)

.unbind()方法允许将选择器中选择所有元素上的特定事件卸载。参数：一个事件名(string)，某个已在元素上绑定好的函数(function)(可选)，返回object。

如有以下html结构:

<div id="clickMe" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html </div>

先给id为'clickMe'的元素绑定单击事件，之后再演示一个卸载事件(.unbind(eventName))的使用方法.

代码如下:

zZ('#clickMe').bind('click',function(){
    var innerhtml = zZ(this).html();
    alert(innerhtml);
});

若要卸载以上事件，可以这么做：

代码如下:

zZ('#clickMe').unbind('click');

也可以为一组元素取消事件绑定：

<div class="allClick" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html(1) </div>
<div class="allClick" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html(2) </div>
<div class="allClick" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html(3) </div>
<div class="allClick" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html(4) </div>
<div class="allClick" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html(5) </div>
<div class="allClick" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html(6) </div>

先给class为'allClick'的一组元素绑定单击事件，之后再演示一个卸载事件(.unbind(eventName))的使用方法.

zZ('.clickAll').bind('click',function(){
    var innerhtml = zZ(this).html();
    alert(innerhtml);
});

若要卸载以上事件，可以这么做：

代码如下:

zZ('.clickAll').unbind('click');

如果一个元素上绑定了多个相同的事件，并要将所有这些事件取消绑定：

<div id="clickMe" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html </div>

先给id为'clickMe'的元素绑定多个单击事件，之后再演示一个卸载事件(.unbind(eventName))的使用方法.

zZ('#clickMe').bind('click',function(){
    alert(1);
}).bind('click',function(){
    alert(2);
}).bind('click',function(){
    alert(3);
});

若要卸载以上所有事件，可以这么做：

代码如下:

zZ('#clickMe').unbind('click');

按需卸载:

如有以下元素：

<div id="clickMe" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 点击我弹出本元素内的html </div>

先给id为'clickMe'的元素绑定多个单击事件，之后再演示一个卸载事件(.unbind(eventName))的使用方法.

zZ('#clickMe').bind('click',function(){
    alert(1);
}).bind('click',function(){
    alert(2);
}).bind('click',function(){
    alert(3);
});

现在的需求就是只卸载alert(2)的这个方法，要怎么做呢，可能要将绑定方法进行一个小的修改，再用另一种方式将它卸载掉：

重新编写的绑定方法:

var funcOne = function(){
    alert(1);
}

var funcTwo = function(){
    alert(2);
}

var funcThree = function(){
    alert(3);
}

zZ('#clickMe').bind('click',funcOne).bind('click',funcTwo).bind('click',funcThree);

下面我就可以将'funcTwo'卸载掉了：

zZ('#clickMe').unbind('click',funcTwo);

另外两个方法不会被移除，只是移除了funcTwo的alert(2).
