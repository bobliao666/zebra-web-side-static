zT.joinArr(arrayOne,arrayTow)  Or  zZ.fn.joinArr(arrayOne,arrayTow)

传入两组数组，将两组数组合并成一个数组，返回合并后的数组

说明:

已知有如下javaScript代码：

function letsBeTogether(){
    var arrayOne  =  ["a","b","c","d","e","f","g","h"]
    var arrayTow  =  ["A","B","C","D","E","F","G","H"]

}

以上代码中有arrayOne和arrayTow两个数组，现在需要将两个数组合并成一个数组，代码如下：

function letsBeTogether(){
    var arrayOne  =  ["a","b","c","d","e","f","g","h"]
    var arrayTow  =  ["A","B","C","D","E","F","G","H"]
    var arrayThree  =  zT.joinArr(arrayOne,arrayTow);
    for(var i=0;i<arrayThree.length;i++){
        alert("合并后_当前下标是："+i+" 当前值是:"+arrayThree[i]);
    }

}

上文代码中arrayThree的输出结果为：

["a","b","c","d","e","f","g","h","A","B","C","D","E","F","G","H"]

从输出结果来看，我们可以使用zT.joinArr()来合并任意两个数组中的元素。
