zT.delArr(Array,Value)  Or  zZ.fn.delArr(Array,Value)

传入数组，和您希望从该数组中移除的项目值，返回已将目标项目移除后的数组。

说明:

已知有如下javaScript代码：

function theArrayTest(){
    var array = [1,2,3,4,5,6,7,8]
}

要求将array中值为'3'的项目移除：

function theArrayTest(){
    var array = [1,2,3,4,5,6,7,8]
    var array = zT.delArr(array,3);//此处执行delArr()方法返回数组后，其值为[1,2,4,5,6,7,8]
}

通过以上代码将‘array’数组中值为‘3’的项目移除，并将移除后的结果返回重新接收。
