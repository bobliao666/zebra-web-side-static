zT.trim(string)  Or  zZ.fn.trim(string)

传入字符串，去除前后空格并返回结果。

说明:

已知有如下javaScript代码：

function letsBeTogether(){
    var str  =  "  goodJob!  ";

}

将str中的字符串去除前后空格:

function letsBeTogether(){
    var str  =  "  goodJob!  ";
    var str  =  zT.trim(str);
    alert(str);

}

以上代码中，使用zT.trim(str)将字符串中的前后空格去除。
