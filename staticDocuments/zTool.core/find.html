.find()

下级选择器，可以在已经找到的元素里继续使用一级选择器的语法查找元素。

说明:

已知有如下html结构代码：

<div id="iAmDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>
<div id="iAmDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>
<div id="iAmDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>
<div id="iAmDiv" mydiv='div' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了正确的元素 </div>

<div id="notDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>
<div id="notDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>
<div id="notDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>
<div id="notDiv" mydiv='div' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 错误 </div>

要求找到自定义属性为mydiv='div'的元素，但id一定是"iAmDiv"，可以通过如下方式筛选想要的元素：

function findMyDivInIAmDiv(){
    zZ("#iAmDiv").find(":mydiv=div");
}

通过以上代码你可以找到id为"iAmDiv"并且属性为 mydiv='div'的元素，但是填入选择器的语法需要注意一下：

1.以冒号开始；

2.冒号之后紧跟属性名称；

3.属性名称后面紧跟等号；

4.等号后紧跟属性的值；

5.属性值请勿使用引号包围；

如下所示：

<div 属性="属性值" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素 </div>
zZ(':属性=属性值');

NOTE:该选择器除了可以查找自定义属性，也可以查找非自定义属性，例如id,class

该下级选择器也可以多次查找，代码如下所示:

<input type="text" class="data-goodOne" value="good" />
<input type="text" value="no" />
<input type="button" value="no" />

function findInput(){
    var thatValue = zZ("<input>").find(":type=text").find(":class=data-goodOne").val();
    alert(thatValue);
}

通过以上示例可见，凡是在一级选择器中适用的匹配规则，在.find()中也同样适用，具体语法请参看"zTool核心/基本/选择器"/"zZ(object)"
