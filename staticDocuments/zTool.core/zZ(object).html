zZ(Object)

简述：zTool的选择器，可以通过HTML元素的id,tagname,class,任何属性，以及自定义属性将元素载入到zTool中，以进行下一步操作。

概要:

在开始操作任何元素之前，我们需要通过一些方式取得元素。

在传统的DOM编程里，可以使用如下方式去查找到我们需要的元素：

1.document.getElementById('id');//此处为获取某id的元素

2.document.getElementsByName('name');//此处为获得某name='name'的元素

3.document.getElementsByTagName('div');//此处为获得tagName的方法

在zTool里可以使用如下方式将元素载入zTool之中：

1.zZ('#id');//此处为获取某id的元素

2.zZ('id');//此处为获取某id的元素

3.zZ('.class');//此处为获取某class的元素

4.zZ(':属性=属性值');//此处为获取某自定义属性的元素

5.zZ('<tagName>');//此处为获取某tagName的元素

6.zZ(Object);//此处直接将对象载入zTools

笔者将在下文中详细介绍以上六种取得元素的方法：

zZ('#id') 和 zZ('id'):

已知有如下html结构代码：

<div id="iAmDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了id为iAmDiv的元素 </div>

要找到id为"iAmDiv"的div,使用如下代码便可载入元素：

zZ('#iAmDiv');//通常情况下，目标元素id前加井号可以让zTool查找并载入这个元素
zZ('iAmDiv');//目标元素id前不加井号也可以找到

要找到id为"iAmDiv"的div,并且alert出其中内容的完整代码是：

function FindiAmDiv(){
 var content = zZ('#iAmDiv').html();//.html()方法用于输出其标签之间的内容,详情请参考api的“html对象操作”/“.html()”
 alert(content);
}

你找到了id为iAmDiv的元素

获得class为"iAmDiv"的一组元素对其操作，取得元素数组，并循环输出其中内容:

function FindClassiAmDiv(){
 var iAmDivArr = zZ('.iAmDiv').Elements();//.Elements()是将查找到的所有元素以数组的方式输出的方法,详情请参考api的“zTool核心/基本/选择器”/“.Elements()”
 for( var i=0 ;i<iAmDivArr.length ; i++ ){
  alert(iAmDivArr[i].innerHTML);
 }
}

当然，获得其数组中第一个元素并对其操作就不需要那么麻烦了:

function FindClassiAmDivOne(){

}

zZ(':属性=属性值'):

已知有如下html结构代码：

<div divId="myDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素 </div>

获得标有自定义属性divId="myDiv"的元素，并alert标间之中的内容:

function FindmyDiv(){
 var content = zZ(':divId=myDiv').html();//.html()方法用于输出其标签之间的内容,详情请参考api的“html对象操作”/“.html()”
 alert(content);
}

通过以上代码你可以找到属性divId="myDiv"的元素，但是填入选择器的语法需要注意一下：

1.以冒号开始；

2.冒号之后紧跟属性名称；

3.属性名称后面紧跟等号；

4.等号后紧跟属性的值；

5.属性值请勿使用引号包围；

如下所示：

<div 属性="属性值" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素 </div>
zZ(':属性=属性值');

NOTE:该选择器除了可以查找自定义属性，也可以查找非自定义属性，例如id,class

属性选择器也可以像.class选择器一样，载入并操作元素数组：

<div divId="myDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素1 </div>
<div divId="myDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素2 </div>
<div divId="myDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素3 </div>
<div divId="myDiv" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 你找到了divId="myDiv"的元素4 </div>

找到所有的属性divId="myDiv"的元素，并循环alert出其中的内容：

function FindallmyDiv(){
 var myDiv = zZ(':divId=myDiv').Elements();//.Elements()是将查找到的所有元素以数组的方式输出的方法,详情请参考api的“zTool核心/基本/选择器”/“.Elements()”
 for( var i=0 ;i<myDiv.length ; i++ ){
  alert(myDiv[i].innerHTML);
 }
}

假设我需要找到其html/body下所有的div：

zZ('<div>');

或者如果我想要找到html/body下所有的span：

zZ('<span>');
