.html()

向查找到的所有节点内设置innerHTML,或者获取所查找到第一元素之innerHTML。

说明:

已知有如下html结构代码：

<div id="getThisInnerHtml" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >获取正确 </div>
<div style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >错误 </div>
<div style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >错误 </div>
<div style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >错误 </div>
<div style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >错误 </div>

要求找到id为‘getThisInnerHtml’的div,并alert出其内部html文本，代码如下：

function getThisInnerHtml(){
    var innerHTML  =  zZ("#getThisInnerHtml").html();
    alert(innerHTML);
}

当然，也可以设置其元素内的html值,如有以下html结构：

<div class='box' id="setThisInnerHtml" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >设置我 </div>
<div class='box' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >设置我 </div>
<div class='box' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >设置我 </div>
<div class='box' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >设置我 </div>
<div class='box' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >设置我 </div>

要求找到id为‘setThisInnerHtml’的div,并将其内部html替换成"设置成功"：

function setThisInnerHtml(){
    zZ("#setThisInnerHtml").html('设置成功');
}

注意，如果您想设置或替换一组元素的innerHTML当然也可以这么做：

function setBoxInnerHtml(){
    zZ(".box").html('设置成功');
}
