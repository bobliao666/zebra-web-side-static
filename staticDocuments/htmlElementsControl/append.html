.append()

此方法允许您将一个或者一组html节点动态添加至某个节点之下。

说明:

已知有如下html结构代码：

<div id="appendToHere" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" ></div>

要求找到id为‘appendToHere’的div,并向其内部增加一个span元素id为'good'内容为'泥嚎'：

function appendToHere(){
    zZ("#appendToHere").append('<span id="good" >泥嚎 </span >');
}

这时，可以使用.html方法查看其结果：

function appendToHere(){
    alert(zZ("#appendToHere").html());
}

当然也可以使用此方法为某个元素内增添一组元素，或者多个嵌套元素.

如有以下html结构代码:

<table id="appendMore" border='1' ></table>

要求找到id为‘appendMore’的table,并向其内部增加一组thead,tbody,tr,td和th,代码如下：

function appendMore(){
    var inputHtml = '<thead>'
            +'<th>标题1 </th>'
            +'<th>标题2 </th>'
            +'<th>标题3 </th>'
            +'<th>标题4 </th>'
            +'<th>标题5 </th>'
            +'</thead>'
            +'<tbody>';
    for(var i=0;i<10;i++){
        var inputHtml += '<tr>'
                +'<td>列1 </td>'
                +'<td>列2 </td>'
                +'<td>列3 </td>'
                +'<td>列4 </td>'
                +'<td>列5 </td>'
                +'</tr>'
    }
    inputHtml+='</tbody>';
    zZ("#appendMore").append(inputHtml);
}

以上代码将会组合一个完整的表格。
