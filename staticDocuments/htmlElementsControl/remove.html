.remove()

移除其选择器所选择到的所有元素。

说明:

已知有如下html结构代码：

<div class='removeAll' id="removeThis" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 将被移除 </div>
<div class='removeAll' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 将被移除 </div>
<div class='removeAll' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 将被移除 </div>
<div class='removeAll' style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" > 将被移除 </div>

要求找到id为‘removeThis’的div,并将其从dom树中直接移除：

function removeThis(){
    zZ("#removeThis").remove();
}

或者将一组元素从dom树中移除:

function removeAll(){
    zZ(".removeAll").remove();
}
