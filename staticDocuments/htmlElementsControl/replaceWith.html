.replaceWith()

此方法允许您将选择器中的多个元素替换为某元素。

说明:

已知有如下html结构代码：

<div class="replaceAll" id="replaceMe" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>

要求找到id为‘replaceMe’的div,并用一个id为'iAmDiv'的元素替换其‘replaceMe’：

function replaceMe(){
    zZ("#replaceMe").replaceWith('<div id="iAmDiv" >我替换了id为‘replaceMe’的div </div >');
}

结果将会如下:

<div id="iAmDiv" >我替换了id为‘replaceMe’的div </div >
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>
<div class="replaceAll" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>

或者在以上结果里，将class为'replaceAll'的元素替换为span：

function replaceAll(){
    zZ(".replaceAll").replaceWith('<span>我替换了‘replaceAll’ </span >');
}

其结果将会如下:

<div id="iAmDiv" >我替换了id为‘replaceMe’的div </div >
<span>我替换了‘replaceAll’ </span >
<span>我替换了‘replaceAll’ </span >
<span>我替换了‘replaceAll’ </span >
<span>我替换了‘replaceAll’ </span >
