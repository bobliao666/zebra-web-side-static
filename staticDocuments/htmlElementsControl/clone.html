.clone()

此方法可以将选择器中的第一位元素,在内存中创建一个副本，不予dom树中的任何一个对象产生引用。

说明:

已知有如下html结构代码：

<div id="cloneMe" style="border:1px solid #000; background-color:#fff; width:60px; height:60px; color:#000" >我将被替换</div>

要求找到id为‘cloneMe’的div,并将其在内存中创建一个副本：

function cloneMe(){
    var cNode = zZ("#cloneMe").clone();
}

上文便是将id为'cloneMe'的div在内存中创建了一个副本叫'cNode',因为之间无任何引用，cNode将和dom中的'cloneMe'节点的双等号判断为false:

function cloneMe(){
    var cNode = zZ("#cloneMe").clone();
    var tNode = zZ("#cloneMe").E();

    tNode == cNode;//false
    tNode === cNode;//false
    tNode == zZ("#cloneMe").E();//true
    cNode == zZ("#cloneMe").E();//false
    zZ("#cloneMe").clone() == zZ("#cloneMe").E();//false
}

由上可看出，其'cloneMe'元素在内存中的副本，并不等同于其在dom树中的'cloneMe'节点.
